# Personal Portfolio

This repository contains the source material and implementation for a personal portfolio project.

## Repository layout

- `choko/` — Next.js portfolio implementation.
- `doc/` — static export/documentation assets.
- `planning/` — strategy, content, architecture, and execution planning docs.
- Root markdown files — drafts, content sheets, and portfolio source notes.

## Local development (Next.js app)

From `choko/`:

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build production: `npm run build`

## Git workflow recommendation

- Keep commits focused by area (docs, app, planning, assets).
- Use descriptive commit messages.
- Push via `main` branch to `origin`.
